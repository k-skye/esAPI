# 查询考试信息接口文档

##### 请求方式：
HTTPS - GET
##### URL：
```
https://service-gt609p42-1252070599.gz.apigw.tencentcs.com/release/getExamTime?account=学号&password=密码
```
##### 参数：
| 名称      |  说明  |
| :--------| :-----|
| account  | 学号  |
| password | 密码  |

##### 错误返回：
```
{'error':'错误信息'}
```
##### 正确返回：
JSON格式的信息
